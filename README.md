# Superstore Sales Performance Dashboard

## Overview
This project presents an interactive Power BI dashboard built using the Superstore sales dataset. The dashboard provides insights into sales performance, customer segments, product categories, regional sales, and top-performing products.

## Tools Used
- Power BI
- CSV Dataset
- Data Visualization

## Key KPIs
- Total Sales: 2.26M
- Total Orders: 5K
- Total Customers: 793

## Dashboard Features
- Sales Trend Analysis (2015–2018)
- Sales by Category
- Sales by Region
- Sales by Segment
- Top 10 Products by Sales
- Business Insights Summary

## Key Insights
- Technology category generated the highest sales (36.6%).
- Consumer segment contributed the highest revenue.
- West region achieved the highest sales performance.
- Sales increased significantly from 2016 to 2018.
